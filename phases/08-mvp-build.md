# Phase 08 — MVP Build

## Цель

Собрать первый рабочий сайт на Docker, WordPress, теме `football` и плагине `football-data`, но пока на мок-данных.

## Что входит в MVP

- Главная.
- Новость single.
- Игрок single.
- Матч single.
- Букмекеры archive.
- Букмекер single.
- Турнир single.
- Команда single.
- Базовая админка плагина.
- Мок-данные по игрокам, матчам, турнирам, букмекерам.
- Переключатель языка в интерфейсе.

## Что можно отложить после MVP

- Полная генерация всех игроков из API.
- Live polling.
- Расширенные prediction-блоки.
- Глубокие исторические сезоны.
- Сложные сравнения игроков.
- Личный кабинет и избранное.

## Definition Of Done

- Все MVP-страницы открываются.
- Нет PHP ошибок.
- Тема адаптивна.
- Шаблоны визуально близки к референсам.
- Данные приходят через плагин, а не захардкожены в теме.
- README обновлен.

## Результат фазы

- Демонстрационный сайт, который можно показать заказчику.
