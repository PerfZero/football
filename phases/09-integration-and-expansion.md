# Phase 09 — Integration And Expansion

## Цель

После получения API-ключа подключить реальные данные API-Football и расширить сайт за пределы MVP.

## Что делаем

- Вводим API key в админке.
- Проверяем лимиты и тариф.
- Подключаем реальные endpoints по очереди:
  - countries/timezones;
  - leagues;
  - teams;
  - players;
  - fixtures;
  - standings;
  - odds/bookmakers;
  - odds.
- Сравниваем реальные ответы с мок-DTO.
- Исправляем мапперы.
- Настраиваем cron.
- Добавляем массовое создание/обновление страниц.

## Расширение страниц

- Архивы игроков.
- Архивы команд.
- Архивы турниров.
- Архивы матчей.
- Сравнение игроков.
- Расширенные страницы статистики.
- Топы: бомбардиры, ассистенты, карточки.
- Травмы.
- Трансферы.
- Прогнозы.

## Результат фазы

- Сайт работает с реальным API.
- Есть контроль лимитов.
- Есть безопасная синхронизация.
- MVP превращается в полноценную футбольную платформу.
