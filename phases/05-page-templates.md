# Phase 05 — Page Templates

## Цель

Собрать шаблоны WordPress, которые соответствуют нужным типам страниц и могут работать как с мок-данными, так и с данными плагина.

## Обязательные шаблоны MVP

### Home

- Hero.
- Быстрые ссылки: игроки, матчи, турниры, статистика.
- Топ-новости.
- Ближайшие матчи.
- Букмекеры.
- Промо-блоки.

### News Archive

- Список новостей.
- Фильтры по теме/турниру/команде/игроку.
- Пагинация.

### News Single

- Breadcrumbs.
- Hero с изображением.
- Дата, автор, категория.
- Контент.
- Поделиться.
- Главные новости.
- Похожие новости.
- Связанные таблицы/матчи.

### League Single

- Hero турнира.
- Вкладки: обзор, таблица, матчи, команды, игроки, новости.
- Standings.
- Top scorers/top assists.
- Ближайшие и последние матчи.

### Player Single

- Hero игрока.
- Биография.
- Характеристики.
- Позиции на поле.
- Статистика за сезон.
- Карьера.
- Достижения.
- Последние новости.

### Team Single

- Hero команды.
- Состав.
- Матчи.
- Турниры.
- Статистика.
- Новости.

### Match Single

- Scoreboard.
- Статус матча.
- Timeline событий.
- Составы.
- Статистика.
- Odds.
- Новости по матчу.

### Bookmakers Archive

- Hero букмекеров.
- Рейтинг.
- Фильтры.
- Таблица букмекеров.
- Популярные ставки.
- Бонусы.

### Bookmaker Single

- Hero букмекера.
- Рейтинг.
- Бонус.
- Страны.
- Условия.
- Обзор.
- FAQ.
- CTA по партнерской ссылке.

## Результат фазы

- Полный набор PHP templates.
- Мок-данные для каждого шаблона.
- Понятная структура partials.
