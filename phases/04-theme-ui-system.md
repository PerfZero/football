# Phase 04 — Theme UI System

## Цель

Превратить визуальный стиль скриншотов в reusable UI system внутри темы `football`.

## Визуальное направление

- Темная спортивная тема.
- Акцентный зеленый.
- Большие condensed-заголовки.
- Фоновые стадионы/игроки/поле.
- Карточки с темным стеклянным фоном.
- Таблицы, таймлайны, вкладки, рейтинги, иконки.

## Базовые компоненты

- Header/navigation.
- Breadcrumbs.
- Hero section.
- Entity card.
- News card.
- Match card.
- Stats table.
- Standings table.
- Player stat bars.
- Match timeline.
- Lineup panel.
- Odds table.
- Bookmaker rating row.
- Sidebar widget.
- Tabs.
- CTA button.
- Language switcher.

## Страницы-референсы из скриншотов

- Главная: большой hero + топ-новости + ближайшие матчи + букмекеры.
- Новость: hero-статья + sidebar главных/похожих новостей + таблица турнира.
- Игрок: профиль, фото, метрики, вкладки, статистика, карьера, новости.
- Букмекеры: рейтинг, фильтры, таблица, сайдбар с подсказками и бонусами.
- Матч: центр счета, события, составы, статистика, коэффициенты, новости.

## Технический подход

- Вынести цвета, размеры, тени, сетки в CSS variables.
- Сделать PHP partials для повторяемых блоков.
- Для интерактива использовать минимум JS: tabs, filters, load more.
- Подготовить адаптив для mobile/tablet/desktop.

## Результат фазы

- Базовая дизайн-система темы.
- Набор компонентов.
- Макеты главных шаблонов на мок-данных.
